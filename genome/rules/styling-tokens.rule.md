# Token Rule

Use tokens from `@innovaccer/ui-assets/tokens`. Do not use hardcoded hex values, CSS color functions (rgba, oklch, hsla), raw Tailwind color utilities (e.g. `bg-red-500`), or arbitrary spacing values (e.g. `gap-[10px]`). All color, spacing, typography, and elevation values must come from the design token system exported by `@innovaccer/ui-assets/tokens`.
